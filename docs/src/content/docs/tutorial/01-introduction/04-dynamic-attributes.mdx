Just like content interpolation, you can use curly braces to make HTML attributes dynamic.

### Your Task

1.  Update the `<img>` tag to use the `src` variable. You can use the attribute binding shorthand syntax: `<img {src} />`.
2.  Update the `<img>` tag to make the alt text: "`website_name` Logo"
3.  Verify that the PyWire logo appears in the preview.

If the attribute name and the variable name are the same, `{src}` is equivalent to `src={src}`.
