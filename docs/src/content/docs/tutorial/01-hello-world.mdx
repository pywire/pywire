Welcome to the **PyWire Interactive Tutorial**! 

PyWire is a pythonic framework for building reactive web interfaces with minimal boilerplate. In this tutorial, you'll learn the core concepts by building real examples directly in your browser.

### Your Task

Look at the code in the editor. Inside the `---html---` block, change the `<h1>` text from "PyWire" to **"Hello PyWire!"**.

Notice how the preview on the right updates instantly as you type. Once you complete the task, the "Next" button will light up!

```wire
count = wire(0)
---html---
<h1>Count: {count}</h1>
<button @click={$count += 1}>Increment</button>
```
