In PyWire, reactivity is powered by the `wire()` function. 

### Core Concepts

1. **State**: Use `count = wire(0)` to create a reactive variable.
2. **Access**: Access it in HTML using `{count.value}`.
3. **Mutation**: Inside Python functions, use the `$` prefix to mutate state. For example, `$count += 1` automatically triggers a UI update.

### Your Task

Implement the `increment` function so that clicking the button increases the count. Use the **`$count`** syntax to perform the addition.
